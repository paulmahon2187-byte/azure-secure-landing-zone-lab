# Security Findings

This directory contains security findings identified during the assessment of the Azure lab environment.

Findings will document insecure configurations, associated risks, evidence, remediation actions and validation results.

## Finding Structure

Each security finding should include:

- Finding title
- Severity
- Affected resource
- Description
- Security impact
- Evidence
- Recommended remediation
- Remediation performed
- Validation after remediation
- Relevant Microsoft guidance

## Planned Assessments

- Excessive Azure RBAC permissions
- Overly permissive network security group rules
- Public network exposure
- Insecure storage account configuration
- Missing diagnostic settings
- Missing resource logs
- Key Vault access-control weaknesses
- Defender for Cloud recommendations
- Unprotected or unmanaged resources
- Resource deletion and configuration-change monitoring

## Severity Ratings

Findings will be categorised as:

- Critical
- High
- Medium
- Low
- Informational
