# Project Screenshots

This directory contains screenshots used as evidence throughout the Azure Secure Landing Zone and Threat Detection Lab.

Screenshots may demonstrate resource configuration, successful deployment, security findings, remediation and monitoring results.

## Screenshot Guidelines

Before uploading a screenshot:

- Remove or obscure subscription IDs
- Remove tenant IDs where unnecessary
- Remove email addresses and personal information
- Remove public IP addresses unless required as controlled evidence
- Never expose passwords, secrets, access keys or tokens
- Crop unrelated browser tabs and desktop content
- Use clear and descriptive filenames

## Naming Convention

Use descriptive filenames such as:

- `resource-group-created.png`
- `virtual-network-subnets.png`
- `nsg-rules-before-remediation.png`
- `nsg-rules-after-remediation.png`
- `log-analytics-workspace.png`
- `defender-recommendation.png`
- `kql-query-result.png`

## Current Status

Screenshots will be added as implementation and testing progress.
